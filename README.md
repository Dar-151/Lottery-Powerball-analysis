# Lottery-Powerball-analysis
Analysis of New York Lottery Powerball winning numbers

## Project Overview
This project analyzes New York Lottery Powerball winning numbers from 2010 to 2025.
The purpose of the project is to explore patterns in the Powerball data using Python packages and Jupyter Notebook.

## Repository Contents
- Final project.ipynb  
  Jupyter Notebook containing data attributes, cleaning, visualizations, and conclusions.

- Lottery_Powerball.csv  
  Dataset used for the Powerball analysis.

- README.md  
  Overview and description of the project and repository files.
  
- New York Powerball Paper.docx
  A one page paper explaining the analysis on the powerball.

- New York Lottery Analysis(1).pptx
  A slide show for presenting the information I found from doing the analysis
